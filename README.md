# Teratheme QA 검수 센터

브랜치별 확정 디자인 스크린샷 + 검수 체크리스트

## 사용법

1. **PR 머지 전** — `/qa-capture` 실행 → 자동으로 스크린샷 캡처 + Issue 생성
2. **검수자** — Issue에서 체크리스트 확인 후 체크
3. **미통과 시** — 수정 이슈 자동 생성 → 새 브랜치

## 라벨

| 라벨 | 의미 |
|------|------|
| `QA: 대기 중` | 머지 완료, 검수 대기 |
| `QA: 검수 중` | 검수 진행 중 |
| `QA: 통과` | 모든 항목 통과 |
| `QA: 미통과` | 수정 필요 항목 있음 |
